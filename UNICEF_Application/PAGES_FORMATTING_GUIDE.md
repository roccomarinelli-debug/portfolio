# PAGES TWO-COLUMN CV FORMATTING GUIDE
## Create a Professional UNICEF-Branded CV in 15-20 Minutes

---

## STEP 1: SET UP DOCUMENT (2 minutes)

1. **Open Pages** on Mac
2. **Create New Document** → Choose "Blank" template
3. **Document Setup:**
   - Click "Document" button (top right)
   - Set margins: **2cm all sides**
   - Page size: **A4** (standard)
   - Orientation: **Portrait**

---

## STEP 2: CREATE HEADER SECTION (3 minutes)

### Your Name
1. Type: **ROCCO MARINELLI**
2. Format:
   - Font: **Helvetica Neue Bold** or **Arial Black**
   - Size: **32pt**
   - Color: **#002E5D** (UNICEF Dark Blue)
   - Alignment: **Left**
   - Tracking: **Tight** (reduce letter spacing slightly)

### Subtitle
1. Press Enter, type: **Digital Content Strategist | Education-Focused Communication Specialist**
2. Format:
   - Font: **Helvetica Neue Regular**
   - Size: **12pt**
   - Color: **#4a5568** (Medium gray)
   - Alignment: **Left**

### Accent Bar
1. Insert → Shape → Line
2. Draw horizontal line under subtitle
3. Format line:
   - Width: **8pt** (thick)
   - Color: **#1CABE2** (UNICEF Primary Blue)
   - Length: **Full width of page**
4. Add space after (Enter 2-3 times)

---

## STEP 3: CREATE TWO-COLUMN LAYOUT (5 minutes)

### Set Up Columns
1. Select all text below the header (everything except name/subtitle/line)
2. Go to **Format** → **Layout**
3. Set **Columns: 2**
4. **Column spacing: 1.5cm**
5. **Left column width: 30%**
6. **Right column width: 70%**

**Alternative Method (More Control):**
1. Insert → Text Box for left column
   - Width: 30% of page
   - Position: Left aligned
2. Insert → Text Box for right column
   - Width: 70% of page
   - Position: Right aligned
3. Link text boxes if content flows

---

## STEP 4: FORMAT LEFT COLUMN (5 minutes)

### Section Headers (CONTACT, TECHNICAL SKILLS, etc.)
- Font: **Helvetica Neue Bold**
- Size: **11pt**
- Color: **#002E5D** (Dark Blue)
- ALL CAPS
- Add left border: 3pt, **#1CABE2** (Primary Blue)
- Spacing: **12pt before, 6pt after**

### Body Text
- Font: **Helvetica Neue Regular**
- Size: **9pt**
- Color: **#1a202c** (Almost black)
- Line spacing: **1.2**

### Bullet Points
- Use custom bullets: **•** (bullet character)
- Color: **#1CABE2**
- Indent: 0.3cm

### Education Entries
Make degree names **bold**, institutions regular weight

---

## STEP 5: FORMAT RIGHT COLUMN (5 minutes)

### Main Section Headers (PROFESSIONAL SUMMARY, KEY ACHIEVEMENTS, etc.)
- Font: **Helvetica Neue Bold**
- Size: **14pt**
- Color: **#002E5D**
- Left border: **6pt, #1CABE2**
- Padding left: **0.5cm**
- Spacing: **18pt before, 10pt after**

### Job Titles
- Font: **Helvetica Neue Bold**
- Size: **12pt**
- Color: **#002E5D**

### Company Names
- Font: **Helvetica Neue Semibold**
- Size: **11pt**
- Color: **#1CABE2** (Primary Blue)

### Dates/Duration
- Font: **Helvetica Neue Italic**
- Size: **9pt**
- Color: **#4a5568** (Medium gray)

### Body Text & Bullets
- Font: **Helvetica Neue Regular**
- Size: **10pt**
- Color: **#1a202c**
- Line spacing: **1.3**
- Bullets: **•** in **#1CABE2**

### Special Callouts (⭐ markers)
Replace ⭐ with:
- Background: **#FFC107** (Accent yellow)
- Text: **EDUCATION SECTOR**
- Font: **Bold, 8pt, White**
- Padding: **2pt all sides**
- Border radius: **3pt**

### How to Apply Custom Colors in Pages
1. Select text
2. Click color picker
3. Choose "Color Picker" tab
4. Enter hex code (e.g., #1CABE2)
5. Save to swatches for reuse

## PRO TIPS

### Consistency is Key
- Pick ONE font family (Helvetica Neue recommended)
- Use font weights for hierarchy (Regular, Semibold, Bold)
- Consistent spacing throughout
- Same bullet style everywhere

### Less is More
- Don't use every color
- Primary blue for accents only
- Dark blue for headers only
- Yellow for special callouts only (sparingly)
- Most text should be black/dark gray

### Test Print
Before finalizing:
1. Print one page to test
2. Check if colors are too light/dark
3. Verify text is readable
4. Ensure spacing looks good

### Mobile-Friendly
Remember: PDFs are often viewed on phones
- Don't use fonts smaller than 9pt
- Ensure good contrast
- Test on your phone after creating

---

## TROUBLESHOOTING

**Problem:** Columns won't align properly
**Solution:** Use text boxes instead of column feature for more control

**Problem:** Colors look different in PDF
**Solution:** Use RGB color mode, not CMYK

**Problem:** File size too large
**Solution:** Reduce image quality slightly, ensure no embedded images

**Problem:** Text overflows page
**Solution:** Reduce font size slightly or tighten line spacing

**Problem:** Looks cluttered
**Solution:** Remove less important details, increase spacing
